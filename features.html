Offline

A really cool feature about service workers is that they allow pages and images to work offline!

As a dev: To use this feature you have install your service worker (if using BuckleScript you can use this handy library).

As a user/dev: To see this feature in action you we've set up the service worker on this domain to use this page. Follow the steps below

  1. Make sure the service worker was installed by visiting one of the demo pages
  2. Set your Browser Offline (Airplane mode works too or see image below)
    • It should look very similar to the above when you're done
  3. Now refresh!
  4. You should still be able to see this page AND both images.
    • This picture is hosted via the github website. So if you saw it before reload it's still here

For your projects it may be good to use Workbox as this caching can be get a bit dicey

Is this Page out of date?

If this page doesn't look up to date - please unregister the service worker on a demo page and then come visit here again! This is due to the offline feature of service workers. Not all service workers function this way but for our simple demo this one does
